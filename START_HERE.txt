================================================================================
                    FAST START (COPY & PASTE)
================================================================================

1. INSTALL DEPENDENCIES:
   pip install -r requirements.txt

2. TEST ON YOUR VIDEO (SIMPLE):
   python test_video.py your_video.mp4

3. TEST WITH CUSTOM GENRE:
   python test_video.py your_video.mp4 action 15

That's it! Your thumbnails will be saved in: your_video_thumbnails/
